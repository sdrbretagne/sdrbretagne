Please note: WFM mode doesn't work yet - if you want to use the node for FM DX, please use NFM 48kHz.

To use the node for FM DX: First set the mode to NFM 48khz, then set your PC volume to 0 (turn it all the way down)

Then click on the [NB] button ( Noise Blanker) and slowly turn your volume up.

(Sometimes you need to click on [NB] twice, so if nothing happens click again.

Any filter higher than 48 Khz will not work.

The reason is that we have a very limited internet upload bandwidth, unfortunately :-(

You may occasionally experience some hiccups, if that's the case just leave the node and come back a few minutes later.
